# text objects

| Keybinding | Operation       |
| ---------- | --------------- |
| aw         | a word          |
| iw         | inner word      |
| aW         | a WORD          |
| ap         | a paragraph     |
| ip         | inner paragraph |
| ab         | a bracket       |
| ib         | inner bracket   |
| at         | a tag block     |
| it         | inner tag block |
